# 04 – Asset Checklist (Peitho)

## Core assets (needed before any public launch)

- [ ] Landing page with clear hero headline for primary ICP
- [ ] 60–90 second demo video showing a real Peitho session
- [ ] 3–5 screenshots of core app flows
- [ ] Tagline (under 10 words)
- [ ] Short description (80–120 characters) for directories
- [ ] Long description (250–300 words) for Product Hunt and listings
- [ ] Logo (SVG + PNG, light and dark versions)
- [ ] Social card / OG image (1200x630)
- [ ] About page
- [ ] Privacy policy and Terms of Service

## Alpha assets

- [ ] Personal outreach DM template (friendly, 3–4 sentences)
- [ ] Feedback form or Typeform for alpha users
- [ ] Onboarding walkthrough (manual or in-app)

## Beta assets

- [ ] Waitlist or signup page live
- [ ] Welcome email sequence (Day 0, Day 3, Day 7)
- [ ] Community post template for Indie Hackers / Peerlist
- [ ] 2–3 social posts (founding story + demo)

## v1 Big Push assets

- [ ] Product Hunt: tagline, short description, long description, gallery images, demo GIF
- [ ] Maker first comment (pre-written, post on launch day)
- [ ] Launch tweet / LinkedIn post thread (pre-scheduled)
- [ ] Press kit (logo, screenshots, founder bio, company description)
- [ ] 1 case study or testimonial from alpha/beta user
- [ ] Directory submission copy for 10+ platforms (from 03-channels)
