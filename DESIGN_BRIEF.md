# Abacus Website Redesign - Design Brief

## Strategic Direction

### Core Message
"Master high-stakes conversations with AI-powered practice and peer sparring"

### Target Audience
- Young professionals (22-40) preparing for job interviews, tough conversations, negotiations
- Ambitious communicators wanting to improve influence and persuasion skills
- Learners interested in behavioral psychology and communication techniques
- Competitive users interested in skill progression and leaderboards

## Brand Evolution

### Current State
- Purple/gold/tan color scheme
- Peitho branding (legacy name)
- Single-page static HTML site
- Modern but minimal design

### New Direction (TBD by Plan agent)
- [Awaiting plan recommendations]
- Likely: Modern, dynamic, inspiring visual language
- Should communicate: AI-powered, practical, skill-focused, community-driven

## Feature Pillars

### 1. Hero/Product Showcase
- What is Abacus? (Conversation gym, not generic AI app)
- Core differentiator: Practice + AI + Peer Sparring + Progression
- Interactive demo or video showing practice session
- Clear value prop: "Practice any difficult conversation"

### 2. Scenarios Library
- Show the 6+ conversation scenarios (job interview, difficult colleague, negotiation, etc.)
- Interactive showcase where users can preview scenarios
- Technique cards for each scenario type

### 3. Community & Social Proof
- Leaderboards (skills, streaks, matches)
- User testimonials and success stories
- Public profile pages
- Community highlights / featured users

### 4. Learning Center / Docs
- Technique library (47 concepts)
- Guides: "How to handle tough questions" etc.
- Psychology fundamentals
- Blog/articles about communication
- Video tutorials or case studies

### 5. Pricing & Monetization
- Freemium or subscription model explanation
- Feature comparison table
- Value justification
- Sign-up/download CTA

### 6. App Download
- App Store + Google Play buttons
- Platform descriptions (iOS/Android)
- System requirements
- Reviews/ratings

## "WOW" Interactive Elements

### Must-Have
- [ ] Smooth scroll animations (parallax, fade-in, reveal)
- [ ] Interactive hero with animated elements
- [ ] Live leaderboard or stats counter
- [ ] Technique cards with hover effects
- [ ] Scenario preview (video or animated mock)
- [ ] Dark mode with smooth transitions
- [ ] Mobile responsiveness with adaptive layouts

### Nice-to-Have
- [ ] Recording/playback widget showing conversation practice
- [ ] AI conversation preview (actually run small Claude prompts to show capability)
- [ ] 3D or WebGL effects
- [ ] Microinteractions on CTAs
- [ ] Personalized content based on scroll/interest
- [ ] Easter eggs or hidden animations

## Technical Approach

### Tech Stack (TBD)
- Likely: Next.js 14+ with TypeScript + Tailwind CSS
- Reason: Fast to market, SEO-friendly, component-based, great DevX

### Content Strategy
- Markdown-based docs content
- Strapi or headless CMS for blog (optional)
- JSON for scenarios, techniques, testimonials
- MDX for rich content pages

### Performance Targets
- Lighthouse score: 90+
- Core Web Vitals: Green across board
- Page load: <3s on 4G
- Accessibility: WCAG 2.1 AA

## Color Palette Themes

### Option A: Bold Modern (High Contrast)
- Primary: Deep indigo or navy
- Accent: Electric cyan or vibrant teal
- Neutral: Clean whites and grays
- Vibe: Energetic, tech-forward, trustworthy

### Option B: Premium Warm (Current Evolution)
- Primary: Warm gold or bronze
- Accent: Deep plum or eggplant
- Neutral: Cream and warm whites
- Vibe: Premium, sophisticated, inspiring

### Option C: Dynamic Gradient (Risk)
- Primary: Gradient from purple to blue
- Accent: Neon pink or orange
- Neutral: White to gray
- Vibe: Modern, dynamic, attention-grabbing

## Typography
- Display/Headlines: Bold, geometric sans-serif (e.g., Space Grotesk, Syne)
- Body/Content: Clean, readable sans-serif (e.g., Inter, DM Sans)
- Data/Stats: Monospace for leaderboards and numbers
- Serif accents: Optional for testimonials or blog
